# Workflow Plan - Task 12

## Goal
Implement the Correctness Agent Adapter for Vertex AI Gemini, grounded against the discovered Source-of-Truth (SoT) evidence-plane corpus and aware of the run token/call/cost budget.

## Scope

### In scope
- Implement a budget tracking module `src/gdg_yorku_submission/budget.py`.
- Initialize run budget in `orchestrator.py` `start_run()`.
- Implement a `GeminiClient` wrapper in `src/gdg_yorku_submission/llm/gemini.py` supporting mock and real Gemini LLM calls.
- Implement the correctness specialist agent in `src/gdg_yorku_submission/correctness/agent.py`.
- Integrate the correctness specialist into `src/gdg_yorku_submission/app.py`.
- Write unit and integration tests in `tests/test_correctness_agent.py`.

### Out of scope
- Implementing coordinator and validators (Task 13, 14).
- Implementing security debate loop (Task 20, 21).

## Requirements
| ID | Requirement |
|---|---|
| REQ-001 | Budget tracking: Define RunBudget/BudgetLease, enforce caps, handle lease acquisition and lease checks. |
| REQ-002 | Grounded correctness review: Extract SoT using discover_sot. Run Gemini review using methodology instructions and nonced evidence plane. |
| REQ-003 | No-spec fallback: If SoT not found, set status to `skipped` with reason `no_spec_found_conformance_skipped`. |
| REQ-004 | Output conformance: Schema-validate correctness findings, cap any non-spec findings at medium. |

## Acceptance Criteria
| ID | Requirement | Acceptance Criterion |
|---|---|---|
| AC-001 | REQ-001 | If budget is exceeded, lease acquisition raises BudgetExhaustedError and marks status as failed. |
| AC-002 | REQ-002 | Correctness findings are generated with valid evidence references pointing to the code and SoT. |
| AC-003 | REQ-003 | Ingestion of a no-spec repository results in correctness specialist returning status `skipped` and reason `no_spec_found_conformance_skipped`. |
| AC-004 | REQ-004 | Findings generated by correctness specialist pass validate_correctness_finding and use original-file coordinates. |
